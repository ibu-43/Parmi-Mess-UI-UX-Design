parmi-mess-ui-ux
his is a UI/UX design project for "Parmi Mess," a food ordering app. The design includes a modern and easy-to-navigate interface with an optimized menu layout, combo offers, cart management, and a location page. Created using Figma with a focus on user experience and visual appeal. 🔗 View on Figma

📌 Features
✅ Interactive Menu – Browse and select food effortlessly
✅ Combo Offers – One-click ordering for meal combos
✅ Cart System – Smooth add-to-cart experience
✅ Location Page – Find nearby mess locations
✅ Minimal & Clean UI – Enhancing user engagement

🛠 Tools Used
Figma (Prototyping & Wireframing)
UI/UX Design Principles
Typography & Color Theory
